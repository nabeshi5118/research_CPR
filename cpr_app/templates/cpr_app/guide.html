How to check

---------- CPR Evaluation ----------

Compression Count: Evaluate the count of chest compression

Appropriate Recoil Percent: Percentage of how many of the compression times reach the red line in the plot image.

Appropriate Depth Percent: Percentage of how many of the compression times reach the blue line in the plot image.

Mean Tempo: How many chest compressions per minute can you do if you continue to perform these chest compressions. (BPM)

Appropriate Tempo Percent: Percentage of how many times one set of correct depth and one set of correct release is performed.

Line Graph: Actual sternal compression strength variation

Analyzed Movie: Video after analysis with YOLOv8
